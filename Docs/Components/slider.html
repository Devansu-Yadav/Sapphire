Slider

You can use Range Sliders to specify a numeric value which must be no less than a given value, and no more than another given value. The precise value, however, is not considered important.







Default Range Slider



To add a Default Slider, just use the .range-slider-default class. You can also add the following classes - .range-slider-value, .range-slider-input, .range-slider-field, and .range-slider-field-val.
