Image Component

Images are integral part of the website, useful for profile, logos and carousels. Responsive Image Components are used to display Images on websites, which are screen responsive. They expand upto the width of the container they are in.







Responsive Images



Reponsive images can be used by giving the class name .img-responsive to the img tag. Resize the window, to see the responsiveness of image.






Round Images



Round Images can be used by adding the class .img-round
