Avatar

An Avatar is a graphical representation of a user or the user's character or persona. They can be custom photos uploaded by users, or generated as Identicons as a placeholder.







Profile Avatars



Profile Avatars can be created by adding the .avatar class on the <img> tag.






Sized Avatars



Size of the avatars can be adjusted by choosing either .avatar-lg | .avatar-md | .avatar-sm classes.






Square Avatars



Add square avatars using the .avatar-square class.
