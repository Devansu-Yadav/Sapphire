Lists

Lists are a continuous group of text or images. They are composed of items containing primary and supplemental actions, which are represented by icons and text.







Unordered Lists



The default style of unordered lists is filled circle denoted by the class .ulist-style-default. Other styles of unordered lists can be used by giving the class names .ulist-style-square, .ulist-style-circle, .ulist-style-none to the <ul> tag.

Ordered Lists



The default style of ordered lists is decimal numbers denoted by the class .olist-style-default. Other styles of unordered lists can be used by giving the class names .olist-style-alphabets, .olist-style-roman to the <ol> tag.
